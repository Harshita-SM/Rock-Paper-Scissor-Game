# 🎮 Rock Paper Scissors Game (React)

## 📌 Project Description

This is an interactive **Rock Paper Scissors game** built using **React.js**.
The user plays against the computer, and the app dynamically determines the winner, tracks game progress, and displays statistics.

---

## 🚀 Features

### 🎯 Core Features

* User can choose **Rock, Paper, or Scissors**
* Computer generates a random move
* Winner is displayed instantly

### 📊 Additional Features

* ✅ **Rounds Tracker** – Tracks total rounds played
* 🔄 **Reset Button** – Resets the entire game
* 📜 **Move History** – Displays all previous moves
* 🔥 **Win Streak Tracker** – Tracks consecutive wins

---

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate to the project folder:

```bash
cd rock-paper-scissors
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:5173/
```

*(or the port shown in terminal)*

---

## 🎮 How to Play

1. Click on Rock, Paper, or Scissors
2. The computer will randomly choose a move
3. The winner will be displayed
4. Track your performance using:

   * Rounds counter
   * Move history
   * Win streak
